<h1 align="center">💼 Minu Office - 사내 메신저 프로젝트</h1>

<p align="center">
  개인 프로젝트 입니다!
  사내 메신저처럼 텍스트, 이미지, 파일 전송이 가능한 채팅 애플리케이션입니다. <br/>
  사용자 간 실시간 소통을 위한 TCP 기반 클라이언트-서버 구조로 구현되었습니다.
  C# WPF MVVM 패턴을 이용하여 만들었고, 개인 공부 및 학습용도로 제작하였습니다.
